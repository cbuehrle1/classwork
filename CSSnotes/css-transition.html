CSS Transitions

Normally when you make a css class change to an html element, whether through hover effect or even JavaScript, the process is instantaneous. However using CSS Transitions, won can give time limits, or specifically an transition period between one state and the other.

As stated by MDN, "CSS transitions let you decide which properties to animate (by listing them explicitly), when the animation will start (by setting a delay), how long the transition will last (by setting a duration), and how the transition will run (by defining a timing function, e.g. linearly or quick at the beginning, slow at the end)."

And when you mouse over this paragraph, there is a little example for you.

For more indepth information and reference material, please head to MDN's article as well as their reference list.
