CSS Web Fonts

CSS web fonts allow for a developer to implement fonts into their web that the viewer does not have locally installed.

By linking the reference site for the font, the browser will pull from linked page after searching for the font locally on the users system.

There are multiple different file types that can be used, but be careful, because they are not all supported by every brower!

For additional information, both MDN and Google Fonts were helpful for figuring out syntax details and being a database for font types.
