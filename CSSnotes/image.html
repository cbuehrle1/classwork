Background Images

The CSS background-image property sets one or several background images for an element. The images are drawn on stacking context layers on top of each other. The first layer specified is drawn as if it is closest to the user.

To use a background image, you simply use the property and enter the url for the reference image. This is a great start, however, there are also a few properties that can help with styling that image futher.

A couple helpful ones are:

  • background-position
  • background-size

For additional information, I found both MDN and CSS Tricks to be helpful
